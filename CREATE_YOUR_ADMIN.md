# 🎯 TẠO TÀI KHOẢN ADMIN CỦA BẠN

## ⚡ Cách nhanh nhất:

### Chạy script:
```bash
create-admin.bat
```

Script sẽ hỏi:
1. **Tên của bạn**: Nhập tên thật
2. **Email**: Email công ty/cá nhân
3. **Password**: Mật khẩu mạnh
4. **Xác nhận**: Nhập lại password

### Ví dụ:
```
Tên của bạn: Nguyễn Văn A
Email: admin@mycompany.com
Password: MySecure@2026
Xác nhận password: MySecure@2026
```

---

## ✅ Sau khi tạo:

1. Mở: http://localhost:5173
2. Đăng nhập với email/password vừa tạo
3. Click avatar → Admin Panel
4. Quản lý hệ thống!

---

## 🔐 Password mạnh:

Nên có:
- ✅ Tối thiểu 8 ký tự
- ✅ Chữ hoa + chữ thường
- ✅ Số
- ✅ Ký tự đặc biệt (@, #, $, !)

Ví dụ tốt:
- `Admin@2026!`
- `MySecure#Pass123`
- `TimeManager$2026`

---

**Chạy `create-admin.bat` ngay!** 🚀
